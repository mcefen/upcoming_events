# Proyecto Upcoming Events

Este es un proyecto que muestra los torneos de vidojuegos próximos, utilizando la WEB de F5 Events.

## Descripción

El proyecto "Upcoming Events" está diseñado para mostrar una lista de eventos que están por venir utilizando la WEB de F5 Events.
Proporciona información clave sobre cada evento, como la fecha, título, numero de participantes, descripción...
